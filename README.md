## Getting Started
 
### Installation

1. Clone the repo
   
2. Create a virtual envirnoment
   ```sh
   python<version> -m venv <virtual-environment-name>
   ```
3. Install libraries
  ```sh
    pip install -r requirements.txt
   ```
4. Start the server
   ```js
   uvicorn main:app --reload 
   ```
